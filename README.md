# Maze generator code

# About

A simple maze generator code example written using Typescript and run using Node.js 

## How to run this code

Prerequisites
- Node
- pnpm (or some other package manager like npm)

1. Install dependencies

```
#npm
npm install

#pnpm
pnpm install
```

2. Run the code

```
#npm
npm run start

#pnpm
pnpm start
```
